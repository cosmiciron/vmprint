# draft2final Samples

These are copyable sample source files for `draft2final`.

Current samples:

- [manuscript-sample.md](./manuscript-sample.md)
- [screenplay-sample.md](./screenplay-sample.md)

You can render them directly:

```bash
draft2final manuscript-sample.md --using mkd-manuscript --out manuscript-sample.pdf
draft2final screenplay-sample.md --using mkd-screenplay --out screenplay-sample.pdf
```

These samples are intended to show real `draft2final` syntax, not placeholder pseudo-Markdown.
