# VMPrint Preview Has Moved

The browser preview component now lives in its own repository:

https://github.com/cosmiciron/vmprint-preview

If you came here looking for the browser-side preview, canvas rendering surface, hit-testing, text selection, SVG export, or the standalone preview playground, that is now the correct home.

The standalone repo contains:

- the `@vmprint/preview` package
- the local preview playground
- browser-focused preview and interaction tooling

This repository remains the home of the core VMPrint engine and related packages.
